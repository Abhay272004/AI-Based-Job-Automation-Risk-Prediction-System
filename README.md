🚀 AI-Based Job Automation Risk Prediction System
📌 Overview

This project predicts the automation risk of different job roles using Machine Learning.
With the rapid advancement of AI, many jobs are evolving or becoming automated. This system helps analyze job-related features and classify the risk level into categories such as Low, Medium, High, and Very High.

🎯 Problem Statement

As AI and automation technologies grow, it becomes difficult for individuals to understand which jobs are at risk. There is a need for a system that can:

Analyze job-related data
Predict automation risk
Provide insights for better career decisions
💡 Solution

This project uses a Random Forest Classifier to predict job automation risk based on multiple features such as:

Automation score
Number of automatable tasks
Digital infrastructure index
Occupation type
